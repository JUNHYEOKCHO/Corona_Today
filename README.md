# Coronatoday

한국의 코로나19 데이터를 분석하여 보여주는 웹사이트입니다.

2020년 9월 1일 코로나 수도권 감염이 확산되는 시기에 맞춰 오픈하였습니다.

이 웹사이트는 2가지 타입의 정보를 보여줍니다.

http://coronatoday.co.kr/
(10.21 서비스 중단)
